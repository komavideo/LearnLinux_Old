Ubuntu Server 16 LTS 安装
=========================

## 知识点

* Ubuntu Server下载
* 虚拟机安装
* 系统更新

## Ubuntu Server下载

**下载地址：**

https://www.ubuntu.com/

**注意：**
选择服务器的LTS版本，以获得长期支持。

## 虚拟机安装

在Oracle VirtualBox虚拟机中安装Ubuntu Server 16LTS。

## 系统更新

~~~bash
$ nano up
...
sudo apt-get update
sudo apt-get upgrade
...
chmod +x up
./up
~~~
